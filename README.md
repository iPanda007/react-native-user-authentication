# User Authentication App

A React Native Expo application with user authentication and modern UI.

## Quick Start

### Prerequisites
- [Bun](https://bun.sh/) installed
- [Expo Go](https://expo.dev/go) app on your mobile device

### Installation & Run

```bash
# Install dependencies
bun install

# Start development server
bun run start
```

Then scan the QR code with your mobile device or press:
- `a` for Android
- `i` for iOS  
- `w` for web

## Available Scripts

- `bun run start` - Start development server
- `bun run android` - Run on Android
- `bun run ios` - Run on iOS
- `bun run web` - Run in web browser

## Features

- User authentication (login/signup)
- Swipe navigation
- Tab-based interface
- NativeWind styling
- TypeScript support

## Tech Stack

- React Native 0.81.5
- Expo 54
